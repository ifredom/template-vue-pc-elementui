# 如何使用

```bash
# 克隆项目
git clone https://github.com/ifredom/template-vue-typescript.git

# 安装依赖
yarn install

# 启动服务
yarn serve
```

浏览器访问 http://localhost:9527

## 发布

```bash

# 构建生产环境
yarn build
```
